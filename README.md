# CRM System

---

## ✨ Overview
This is a CRM (Customer Relationship Management) system developed as a university project. The system allows managing employees, orders, and records.

---

## 💡 Key Features
- Register as a company owner and create employee accounts

- Access employee dashboards to monitor activity

- Create service forms specifying name, description, price, and available employees

- Share service forms with clients via unique links

- Clients can view service details, available employees, and their free time for booking

---

## 🏗️ Technology Stack

- **Framework:** ASP.NET Core MVC
- **ORM:** Entity Framework Core
- **Database:** MySQL
- **Languages:** C#, JavaScript, CSS, HTML

---

## 🚀 Quick Start
1. Install .NET SDK and MySQL.
2. Create a database and update the connection string in appsettings.json:
    ```bash
    "DefaultConnection": "Server=localhost;Database=CRM_DB;User=USER;Password=PASSWORD;"
    ```
3. Run the project using command line:
    ```bash
    dotnet run
    ```
4. Go to http://localhost:5000
